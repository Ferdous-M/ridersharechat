## Real-Time Chat System for Ride-Sharing Platform

A scalable one-to-one messaging system designed for ride-sharing applications, enabling real-time communication between riders and drivers.

### Key Features

* Real-time bidirectional messaging using WebSocket and STOMP.
* User-specific message routing for private conversations.
* Persistent message storage with chat history retrieval.
* REST APIs for conversation and message management.
* Secure communication using JWT-based authentication and authorization.

### Technical Highlights

* Implemented low-latency message delivery using Spring WebSocket.
* Built backend services for message persistence and conversation tracking.
* Designed database models to efficiently store and retrieve chat history.
* Integrated real-time and REST-based communication within a single backend architecture.
* Developed using Spring Boot, WebSocket, STOMP, PostgreSQL, JPA/Hibernate, and JWT.

### Tech Stack

Java • Spring Boot • WebSocket • STOMP • PostgreSQL • JPA/Hibernate • JWT • Maven
